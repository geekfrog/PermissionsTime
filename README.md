# PermissionsTime
### 支持跨服的权限限时插件

#### 构建地址:[http://ci.frog.gg/jenkins/job/PermissionsTime/](http://ci.frog.gg/jenkins/job/PermissionsTime/)

##### 插件目标：

黄色标记已完成

- ==在配置文件中编辑权限包(可含有多个权限组和多个权限)==
- ==时间可以累加==
- ==支持重载==
- 命令支持给玩家==添加==、设置、移除、权限包时间
- 支持UUID
- mysql保存数据
- 支持不同世界权限
- 支持前后缀管理
